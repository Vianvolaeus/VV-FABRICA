## Installation

### via Remote Repository

1. Open Blender and go to **Edit > Preferences > Get Extensions**.
2. Click the **Repositories** dropdown and select **Add Remote Repository**.
3. Enter the repository URL: `https://vianvolaeus.github.io/VV-FABRICA/dist/index.json`
4. VV-FABRICA will appear in the extensions list for installation and automatic updates.

## Requirements

- Blender **4.5.0** or later
